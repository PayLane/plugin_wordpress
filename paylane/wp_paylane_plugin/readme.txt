PayLane Secure Form for WordPress.

== Description ==

This plugin gives you opportunity to integrate PayLane payment service with your WordPress site.

== Installation ==

1. Upload `wp_paylane_plugin/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `[paylane_plugin] ` in your posts/pages
